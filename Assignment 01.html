Welcome to My Learning Journey !

This website is created as a complete HTML project for a web design and computer science class. Its purpose is to demonstrate a clear understanding of HTML structure and elements.

Throughout this page, different HTML tags are used such as headings, paragraphs, lists, images, links, navigation, and forms. Each section represents a step in learning how to build a structured and meaningful webpage.

In the following sections, you will learn more about the learning journey, future goals, and how to get in contact.

Main Goal

My name is Sana, a mathematics student who explored different paths in search of a clear goal and direction. Through this journey, I discovered a strong interest in frontend development and web design, where logic and creativity come together to build meaningful digital experiences.

Learning coding feels like playing a thoughtful game. It is about understanding how websites work behind the scenes and how ideas gradually become visible on the screen. Coding helps me stay focused, strengthens my problem-solving skills, and builds confidence with every new concept I learn.
